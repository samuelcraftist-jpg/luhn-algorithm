# Luhn Algorithm (C++)

This project is a simple implementation of the **Luhn Algorithm**, which is used to validate credit card numbers.  
It checks whether a given number passes the checksum formula commonly used in payment systems.


## How to Run
- Compile the file with a C++ compiler (for example, using `g++`).
- Run the generated program.
- Enter a credit card number when prompted, and the program will tell you if it is valid or not.

## Example
Input:

Enter the credit card number: 4111111111111111

output:

 4111111111111111 is valid.



## Features
- Validates credit card numbers using the Luhn Algorithm.
- Simple and lightweight C++ console application.
- Easy to extend for other numeric validation tasks.
